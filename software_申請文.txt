================================================================================
  ソフトウェアインストール申請書
================================================================================

【共通の申請理由】

  業務記録の作成・管理、および画像・文書・コードの確認作業の
  正確性と効率を高めるため。

  いずれも無償のオープンソースで、公式サイトから入手でき、
  インストール後はオフラインで動作するため
  社外へ情報が出る心配はありません。


================================================================================
  ① DB Browser for SQLite（DB4S）
================================================================================

  入手元     : https://sqlitebrowser.org/dl/
  ライセンス : MPL v2 / GPL v3（無償・商用利用可）

  用途
    業務データ（作業ログ・測定データ目録）を管理するSQLiteファイルの
    表構成変更、一括取込、検索など管理作業のため。

  安全性
    オフライン動作。社外へのデータ送信なし。
    インストール不要のZIP（ポータブル）版も選択可能。


================================================================================
  ② sql-asm.js（sql.js のファイル 1 点）
================================================================================

  入手元     : https://github.com/sql-js/sql.js/releases
  ライセンス : MIT ライセンス（無償・商用利用可）

  用途
    自作の HTML 入力ツールがブラウザ（Edge）上で業務データを
    表示・編集するために使う部品ファイル。
    このファイル 1 つを同じフォルダに置くだけで動作。

  安全性
    インストール不要。ブラウザ内のみで完結。
    オフライン動作。社外へのデータ送信なし。


================================================================================
  ③ GIMP（ギンプ）
================================================================================

  入手元     : https://www.gimp.org/
  ライセンス : GPL v3（無償・商用利用可）

  用途
    検査・測定画像の歪み補正や複数画像の連結、色合い補正を行い、
    記録画像の品質と作業の正確性を高めるため。
    手順書用スクリーンショットへの注釈付けにも使用。

  安全性
    オフライン動作。社外へのデータ送信なし。


================================================================================
  ④ WinMerge（ウィンマージ）
================================================================================

  入手元     : https://winmerge.org/
              （日本語版） https://winmergejp.bitbucket.io/
  ライセンス : GPL v2（無償・商用利用可）

  用途
    作業記録文やプログラムの新旧比較により、
    変更箇所と変更漏れを確実に確認し、作業の正確性を高めるため。

  安全性
    オフライン動作。社外へのデータ送信なし。


================================================================================
  ⑤ サクラエディタ
================================================================================

  入手元     : https://sakura-editor.github.io/
  ライセンス : zlib ライセンス（無償・商用利用可）

  用途
    作業記録文・プログラムコードの作成・編集を効率化し、
    文字コードや改行の確認によって記録の正確性を保つため。

  安全性
    オフライン動作。社外へのデータ送信なし。


================================================================================
  ⑥ gvim（GUI版 Vim）
================================================================================

  入手元     : https://www.vim.org/
  ライセンス : Vim ライセンス（GPL 互換・無償・商用利用可）

  用途
    キーボード操作を中心に文書・コードを編集することで、
    作業記録やコード編集を高速化し、操作ミスを減らすため。

  安全性
    オフライン動作。社外へのデータ送信なし。最新版を使用。
